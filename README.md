## Python and ImageJ Macro scripts to pre-process, analyze, and visualize MRI data.

# 1. Register fMRI data in Nifty format to Allen Brain atlas using [ABBA](https://abba-documentation.readthedocs.io/en/latest/) in ImageJ.
# 2. Export ROIs of the brain regions.
# 3. Convert ImageJ ROIs to tif images using the roi_to_mask.ijm script.
# 4. Analyze brain connectivity using abba.py code.
